MiU Theme / View Lifecycle

MiU icinde bir temanin framework'e nasil oturdugunu ve bir request sirasinda view motorunun bunu nasil isledigini gosteren gorsel ozet. Burada iki katman birlikte anlatilir: once tema dosyalari layout + partial + page yapisina oturur, sonra Core\View\View bu dosyalari placeholder, dil ve helper kurallariyla render eder.

Tema Uyarlama Akisi

Template Syntax Ozet

Normal Data

{title}, {admin_name}, {content} gibi veriler page service veya shared view data icinden gelir.

Translation

{{Lutfen alaninizi seciniz}} formatinda yazilir. DB'de yoksa varsayilan dil icin kayit olusturulur, diger diller bos acilir.

Partial Include

{>partials/header} ayni view kokundeki ilgili .html dosyasini mevcut data ile render eder.

Condition

{?admin_logged_in}...{/admin_logged_in} ve {?can:users.edit}...{/can:users.edit} bloklari desteklenir.

URL Helper

{url:/login} ve {asset:/css/app.css} path'leri otomatik URL'ye cevirir.

Raw HTML

RawValue kullanilan alanlar escape edilmeden render edilir. Ozellikle {content} layout icinde bu sekilde tasinir.

Temel Kurallar

Tema once parcalanir

MiU temayi tek parca magic sekilde yonetmez. En saglikli kullanim, temayi layout + partial + page yapisina bolup sonra framework'e yerlestirmektir.

Page once, layout sonra

MiU once page template'ini render eder. Sonra ortaya cikan HTML'i RawValue(content) olarak layout'a enjekte eder.

View kokleri ayridir

Site, admin ve mail taraflari farkli basePath ile calisir. Ama hepsi ayni Core\View\View sinifini kullanir.

Shared data otomatik gelir

app_url, assets_url, locale bilgileri, giris yapan kullanici ve izinler bootstrap asamasinda view nesnesine eklenir.

Ceviriler runtime'da kaydolabilir

{{...}} placeholder ilk goruldugunde veritabanina otomatik kaydedilir. Bu, daha sonra panelden toplu ceviri yapmayi kolaylastirir.

Asset referanslari temanin parcasi olarak kalir

Tema icindeki CSS ve JS linkleri ister oldugu gibi birakilir, ister {asset:/...} formatina cevrilir. Framework bunlari runtime'da dogru URL ile birlestirir.
